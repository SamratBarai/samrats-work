How to Use:

  • Click on the graph to add points
  • Drag points to move them around
  • Right-click on a point to remove it
  • Select different curve types to see various equations
  • Polynomial mode: Add as many points as you want for higher-degree equations
  • Grid Snapping: Toggle to snap points to grid lines
  • Dark Mode: Toggle for dark theme
  • Use "Random Points" to generate sample data
